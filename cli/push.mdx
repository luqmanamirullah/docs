---
title: "Push"
---


For [Push](https://cheatcode.co/push) deployed apps, the `push` named export from `@joystick.js/node`, provides additional context related to the environment your app is currently deployed to.

*   `continent` the continent of the instance the app is currently running on (e.g., `north_america`, `europe`, `south_america`, `oceania`, or `asia`).
*   `instance_token` the unique token for the instance the app is currently running on.
*   `current_version` the current version timestamp (e.g., `2024-01-29T00:10:21Z`) for the app.
