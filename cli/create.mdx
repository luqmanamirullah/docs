---
title: "Create"
---


To create a new Joystick app, from your terminal, run:

```bash
    joystick create <app_name>
   ``` 

## Args

*   `<app\_name>` Required
    
    The name of the app you'd like to create. Joystick will use this value for the folder containing your project and as the `name` field in your `package.json` file.
    

## Options

*   \--r, --release
    
    The Joystick release to use when creating this project. Either `production` (default) or `canary`.
