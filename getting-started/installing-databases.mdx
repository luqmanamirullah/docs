---
title: "Installing Databases"
---


To install one of Joystick's [supported databases](databases/providers), follow the official instructions for the database for your operating system from the list below.

## MongoDB

    Recommended version MongoDB v5
    

### MacOS

To install MongoDB on MacOS, [read the official guide from MongoDB](https://www.mongodb.com/docs/manual/tutorial/install-mongodb-on-os-x/#install-mongodb-community-edition).

### Windows

To install MongoDB on Windows, [read the official guide from MongoDB](https://www.mongodb.com/docs/manual/tutorial/install-mongodb-on-windows/#install-mongodb-community-edition).

### Linux

To install MongoDB on Linux, [read the official guide from MongoDB](https://www.mongodb.com/docs/manual/administration/install-on-linux/).

## PostgreSQL

    Recommended version PostgreSQL v14
    

### MacOS

To install PostgreSQL on MacOS, [read the official guide from PostgreSQL](https://www.postgresql.org/download/macosx/).

### Windows

To install PostgreSQL on Windows, [read the official guide from PostgreSQL](https://www.postgresql.org/download/windows/).

### Linux

To install PostgreSQL on Linux, [read the official guide from PostgreSQL](https://www.postgresql.org/download/linux/).
